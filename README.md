# Student Record Management Web App

A Flask-based web application to manage student records.  
Users can add, view, search, and delete student data with validation and file storage.

## Features
- Add student records
- View all students
- Search student by name
- Delete student records
- Simple UI using HTML & CSS

## Tech Stack
- Python
- Flask
- HTML, CSS
- File Handling

## How to Run
1. Download or clone the repository
2. Open the project folder
3. Install Flask:
   pip install flask
4. Run the app:
   python app.py
5. Open browser and go to:
   http://127.0.0.1:5000
